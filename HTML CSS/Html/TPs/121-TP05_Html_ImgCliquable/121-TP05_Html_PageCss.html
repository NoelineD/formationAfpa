le CSS : Cascading Style Sheets

CSS est un langage basé sur des règles — on définit des règles de styles destinées à
des éléments ou des groupes d'éléments particuliers dans la page.
Par exemple, « Je veux que le titre principal de ma page s'affiche en rouge en gros caractères. »
La règle commence par un sélecteur, l'élément HTML mis en forme.
